Project description - Mon Vieux Grimoire

This OpenClassrooms project led me to develop the back-end of a book rating site. We had to create a server with Express and connect it to a MongoDB database. This involved setting up the server structure and managing the communication between the server and the database. I developed the data models and implemented CRUD (Create, Read, Update, Delete) operations for book and rating management. This required special attention to data security and secure storage. I also implemented a secure authentication system for site users. The project also includes handling image upload and optimization, as well as adding features to rate books and calculate average rating. I also respected the best practices of the Green Code to reduce the ecological footprint of the site. To complete the project, I used Mongoose to model MongoDB data, and followed an MVC (Model-View-Controller) architecture to structure the application.

Node.js
Express.js
MongoDB
